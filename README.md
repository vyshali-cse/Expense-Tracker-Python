# Expense Tracker

## Description
A Python application that helps users track and manage their daily expenses using CSV files.

## Features
- Add new expenses
- View saved expenses
- Store expense records in CSV format
- Simple command-line interface

## Technologies Used
- Python
- CSV Module

## Files
- expense_tracker.py
- requirements.txt

## How to Run

1. Install Python
2. Run:
   python expense_tracker.py
